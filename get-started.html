Get Started with Docker MCP

Step 1

Install Docker Desktop if you haven't already. Docker MCP requires Docker Desktop for the integrated catalog experience.

Step 2

Open Docker Desktop and navigate to the MCP Catalog section. Browse the available MCP servers and select the ones that match your use case.

Step 3

Click the one-click deployment button for each server you want to run. Docker will automatically pull the container images and start the services.

Step 4

Connect your favorite MCP client (Claude, Cursor, VSCode, etc.) to the Gateway MCP Server. The configuration details are provided automatically in Docker Desktop.

Step 5

Start building powerful AI agents with access to the tools you've deployed. Your containers are isolated, secure, and ready to scale.

From zero to AI-powered workflows in under 5 minutes.
